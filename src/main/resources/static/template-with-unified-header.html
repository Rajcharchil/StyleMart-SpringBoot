This is an example of how to use the unified header in any page.

Features Available:

  • ✅ Consistent header design across all pages
  • ✅ Working cart functionality with count
  • ✅ Search functionality
  • ✅ Responsive design
  • ✅ Navigation to all sections

How to use the unified header:

  1. Include the header loader script: <script src="/js/header-loader.js"></script>
  2. The header will automatically load when the page loads
  3. Use window.headerLoader.updateCartCount() to update cart count
  4. Use window.headerLoader.getCart() to get cart data
  5. Use window.headerLoader.saveCart(cart) to save cart data
